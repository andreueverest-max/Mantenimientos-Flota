# Gestión de Mantenimiento de Flota

Esta aplicación te permite gestionar los mantenimientos de tu flota de camiones de manera sencilla, con un diseño similar a Microsoft Dynamics 365 Business Central.

## Cómo empezar (Fácil)

Para arrancar el programa, solo tienes que hacer doble clic en el archivo:
**`Iniciar_Aplicacion.bat`**

La primera vez que lo abras, se descargarán los componentes necesarios automáticamente (esto puede tardar un poco). Después, se abrirá tu navegador con el programa listo para usar.

## Datos de Acceso
- **Usuario:** `admin`
- **Contraseña:** `password123`

## Características principales
- **Cuadro de Mando:** Resumen visual de la flota.
- **Vehículos:** Registro completo de camiones.
- **Mantenimiento:** Pantalla unificada para ver planes y registrar trabajos.
- **Configuración:** Edita marcas, modelos y operarios.

---
*Nota: Todos los datos se guardan en tu navegador local. No necesitas instalar bases de datos externas.*
